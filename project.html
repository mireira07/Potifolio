Projeto de "Cadastro de Clientes" para uma empresa

Projeto feito com JS,HTML e CSS de um site de cadastros. O cliente clica em "Cadastrar Cliente" e vai preencher os requisitos solicitados como: Nome,Email,Celular e Cidade. Terminado de preencher vai clicar em "salvar" para manter o cadastro salvo com sucesso e a empresa saber quem são os clientes cadastrados.

Projeto de automoção de uma "Lâmpada"

Lâmpada que trabalha com automação. Passando mouse em cima da lâmpada ela liga, mesma coisa para desliga-lá. Clicando nos botões também conseguimos fazer o mesmo, porém se clicar duas vezes na lâmpada ela estora.
